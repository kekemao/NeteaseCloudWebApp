# netease-clound-webapps

> 这是一个用VUEJS写高仿网易云音乐的webapp，只实现了小部分功能。


### 技术栈
- 传说中的VUE全家桶(vue vue-router vuex)
- axios
- VUX一个基于Vue2的移动端组件


## 安装运行（Build Setup）

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
